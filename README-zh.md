# 如何私人使用
 1. Clone這個repo(我他媽不知道怎麼翻譯lmao)
 1. 建立隱藏檔 `setup/token.json`，格式如下：
 ```json
 {
     "token": "機器人token",
     "mongourl": {
          "turtleSoup": "指定資料庫的mongodb連結，用來存放海龜湯系統的資料"
     }
 }
 ```
 3. 下載第三方套件
 1. `node .`啟動機器人
